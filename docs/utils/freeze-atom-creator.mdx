---
title: freezeAtomCreator
---

```js
import { atom } from "jotai";
import { freezeAtomCreator } from "jotai/utils";

const createFrozenAtom = freezeAtomCreator(atom);
const objAtom = createFrozenAtom({ count: 0 });
```

不是从现有原子创建冻结原子，
`freezeAtomCreator` 采用原子创建器函数并返回一个新函数。
您不仅可以将其用于 `atom`，还可以用于其他 `atomWith*` 创建器，例如 `atomWithReduer`。
