## 使用

```ts
freezeAtom(anAtom): AtomType
```

`freezeAtom` 采用现有原子并返回一个新派生的“冻结”原子。
原子将被 `Object.freeze` 深度冻结。
查找您无意中尝试更改可能导致意外行为的对象（状态）的错误很有用。
您可以对所有原子使用 `freezeAtom` 来防止这种情况。

## 参数

**anAtom** (required): 你想冻结的原子。

## 示例

```js
import { atom } from "jotai";
import { freezeAtom } from "jotai/utils";

const objAtom = freezeAtom(atom({ count: 0 }));
```
