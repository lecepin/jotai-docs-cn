Jotai atoms 完全可以在 React Native 应用程序中使用而无需任何更改。
我们的目标是始终与 React-Native 100% 兼容。

### 持久化

关于持久化特性，React Native 的具体实现在 [utils bundle 中的 atomWithStorage 函数](../api/utils.mdx#atom-with-storage) 中有详细说明。

### 性能

在您的应用中使用 Jotai 时没有已知的特定开销。 一些库会向存储的数据添加 一些/很多 额外的属性和方法以供实际使用，但 Jotai 的行为不同，您总是在操作几乎无法简化的简单内容。

Jotai 原子架构将鼓励您拆分逻辑和数据，提供最顶级的体验来控制您的每一个渲染（[或准确地说是提交](https://reactjs.org/blog/2018/09/10/introducing-the-react-profiler.html#browsing-commits)) 从而达到最佳性能。

永远记住渲染必须是快速的，将计算逻辑拆分为异步操作。
